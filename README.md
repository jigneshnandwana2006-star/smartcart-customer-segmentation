# SmartCart Customer Segmentation System

## Project Overview
This project uses Unsupervised Machine Learning techniques to segment e-commerce customers based on demographics, purchasing behaviour, spending patterns, and engagement data.

The project helps identify:
- High-value customers
- Budget shoppers
- Loyal customers
- Low-engagement users

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## Workflow
1. Data Preprocessing
2. Missing Value Handling
3. Feature Engineering
4. Outlier Detection
5. Correlation Analysis
6. OneHot Encoding
7. Feature Scaling
8. PCA Visualization
9. Elbow Method
10. Silhouette Score
11. KMeans Clustering
12. Cluster Interpretation

---

## Machine Learning Techniques
- KMeans Clustering
- PCA (Principal Component Analysis)
- StandardScaler
- OneHotEncoder

---

## Key Insights
- High income customers showed higher spending patterns
- Website visits alone did not guarantee high purchases
- Premium customers preferred store and catalog purchases
- Different customer groups required different marketing strategies

---

## Results
The model successfully segmented customers into 4 meaningful groups for business-focused customer analysis.
